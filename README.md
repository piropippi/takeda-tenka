# 武田晴信 天下統一伝（Web版）

マインスイーパーと戦国合戦を組み合わせた、スマートフォン縦画面向けWebゲームです。
第一章「諏訪攻略」の1-1「国境の砦」までプレイできます。

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開きます。

## テストと本番ビルド

```bash
npm test
npm run build
```

ビルド結果は `dist/` に生成されます。

## 1-1実装内容

- 6×6盤面、千野頼満の敵マス3個
- 武田軍総兵士数1,200、敵軍・千野頼満隊700
- 周囲8マスの敵数表示と0マス自動展開
- 武田軍6将の攻撃ローテーション
- 通常ダメージ、5%クリティカル、最低ダメージ10
- 連撃補正、士気蓄積、攻撃・反撃ログ
- 勝利・敗北画面
- PWA用マニフェストとService Worker

攻撃・防御は武将の能力値、兵士数は各軍・部隊の残存兵力として別々に管理しています。
