# 武田晴信 天下統一伝

戦国時代を題材にしたスマートフォン向け2DゲームのUnityプロジェクトです。

現在は開発用の土台のみを収録しています。ゲーム本体、育成、装備、属性、アイテム、ガチャ、敵兵種、軍師などの機能はまだ実装していません。

## 開発環境

- Unity 6.3 LTS（6000.3.20f1）
- 2D
- 対象プラットフォーム: iOS / Android
- 画面方向: 縦固定

## 開き方

1. Unity HubにUnity 6.3 LTS（6000.3.20f1）をインストールします。
2. Unity Hubの「Add project from disk」で、このリポジトリのルートを選択します。
3. `Assets/Scenes/TitleScene.unity` を開きます。

## ディレクトリ

```text
Assets/
├── Data/       ゲームデータ
├── Prefabs/    プレハブ
├── Scenes/     シーン
├── Scripts/    C#スクリプト
└── Sprites/    2D画像
docs/           仕様・設計ドキュメント
```
